# FileBiggener
 a tool that makes files bigger :)
 ### commands:

 "FileBiggener.exe (filepath) (arguments)"

 ### arguments:
 
 "-d" - debiggen/extract the biggened file to its original size and format
 "-b" - biggen the original file
